# HONESTA - CMRIT Lost & Found

A secure, high-integrity lost and found platform specifically designed for the CMRIT community. HONESTA uses AI-powered image analysis and automated verification challenges to ensure items are returned to their rightful owners.

## üöÄ Features

- **CMRIT Secure Gateway**: Restricted access for Students (@cmrithyderabad.edu.in) and Faculty (@cmritonline.ac.in).
- **AI Verification**: Uses Google Gemini to generate specific verification questions based on the item reported.
- **Privacy First**: Public feed shows items in a high-contrast B&W silhouette to prevent casual "browsing" and protect sensitive details.
- **Triumph Handover**: A dedicated Founder Control Panel allows the person who found the item to mark it as returned once identity is verified.
- **Global Sync**: Persistent cloud database ensures your reports and messages are synced across all your devices.
